# 📋 Join

A Kanban-based task management application inspired by Trello.

Join was developed as a collaborative team project by a group of three developers. The goal of the project was to practice agile software development, teamwork, Git workflows and collaborative coding while building a fully functional task management application.

## 🚀 Installation

Clone the repository:

bash git clone https://github.com/mahapiri/join.git 

Navigate into the project folder:

bash cd join 

Install dependencies if required:

bash npm install 

Start a local server:

bash npx live-server
